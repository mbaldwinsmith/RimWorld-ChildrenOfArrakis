# Defs

XMLForge outputs. Keep files grouped by feature set:
- `BiomeDefs`, `TerrainDefs`, `WeatherDefs` for Arrakis environment
- `ThingDefs/*` for items, buildings, apparel, weapons, plants
- `NeedDefs`, `HediffDefs`, `ThoughtDefs`, `PreceptDefs`, `RitualPatternDefs` for hydration, spice, ideology
- `IncidentDefs`, `PawnKindDefs`, `RecipeDefs`, `ResearchProjectDefs`, `JobDefs`, `WorkGiverDefs`, `SoundDefs` for worms, spice blooms, crafting, and jobs
